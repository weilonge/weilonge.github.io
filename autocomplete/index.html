AutoComplete Plugin Demo

Example 1 with linear search

Example 2 with a tree built with initial character.

This demo shows you an auto-complete widget with the tag feature.

The sample dictionary file is "password.json" with about 8000 entries.

The time spent for each searching operation will be shown for the reference.

In addition, Example 2 is faster than Example 1 generally.


Note: please run this demo in a static web server ( i.e. python -m SimpleHTTPServer ) due to the cross-site protection,
or go to the static site: http://weilonge.bitbucket.org/autocomplete
